Find the Early Voltage of an NPN BJT

To find the Early effect voltage VAV_AVA​ of any particular transistor, one can utilize and derive a formula to calculate VAV_AVA​ using this equation for the large-signal model:

IC=ISeVBEVT(1+VCEVA)I_C = I_Se^{\frac{V_{BE} }{V_T} }\left(1 + \frac{V_{CE} }{V_A} \right) IC​=IS​eVT​VBE​​(1+VA​VCE​​)

Or this equation for the small-signal model:

rO=VA+VCEIC≈VAICr_O = \frac{V_A + V_{CE} }{I_C} \approx \frac{V_A}{I_C} rO​=IC​VA​+VCE​​≈IC​VA​​

However, since the 2N2222 NPN transistor is a BJT, a much simpler way to find the Early voltage is by utilizing the slope from the ICI_CIC​-to-VCEV_{CE}VCE​ characteristic curves in the forward active region, as demonstrated by the illustration below.

Regardless of the value of the constant base current IBI_BIB​, all slopes will have the same value of −VA-V_A−VA​ for IC=0I_C = 0IC​=0

For more details on the calculations in Python, here is the script I used for calculating the Early voltages of each increment.

import statistics

input_data = [
    (1.027752e-03, 1.029726e-03),
    (2.028514e-03, 2.032411e-03),
    (3.003599e-03, 3.009369e-03),
    (3.954082e-03, 3.961678e-03)
]

results = []

for y1, y2 in input_data:
    x1, x2 = 4.8, 5.0
    
    m = (y2 - y1) / (x2 - x1)

    b = m * (0 - x1) + y1

    x = (0 - b) / m

    Va = abs(x)
    results.append(Va)

    print(str(Va))
    
print("\nAverage: " + str(statistics.mean(results)))
